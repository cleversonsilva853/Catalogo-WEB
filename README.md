# Cardápio Digital (Delivery Grill)

Sistema de cardápio digital e PDV desenvolvido com Vite, React, TypeScript e Tailwind CSS.

## Tecnologias Utilizadas

- [Vite](https://vitejs.dev/)
- [React](https://react.dev/)
- [TypeScript](https://www.typescriptlang.org/)
- [Tailwind CSS](https://tailwindcss.com/)
- [Supabase](https://supabase.com/)

## Como rodar o projeto localmente

Pré-requisitos: Node.js e npm (ou bun) instalados.

1. Instale as dependências:
```bash
npm install
```

2. Inicie o servidor de desenvolvimento:
```bash
npm run dev
```

3. Abra o navegador no endereço indicado (geralmente `http://localhost:8080`).

## Build para Produção

Para gerar a versão otimizada para produção:

```bash
npm run build
```
